Скрипт для добавления табов на страницу, написанный на чистом JavaScript

Для того, чтобы добавить табы на страницу необходимо изменить имена классов html
на ваши собственные. Делается это в начале скрипта, в объявлении переменных.

Подробное описание. Какая переменная за какой класс отвечает и какую роль играет.

1) "tab": содержит псевдомассив с элементами табов.
2) "info": является родительским классом для табов (используется для
создания делегированного события).
3) "tabContent": содержит псевдомассив контента, который привязан к табам.
